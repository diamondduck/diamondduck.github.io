Try Figure Out what's wrong with notion-app+notion-enhancer

12.11.2020 10:51

Problem

Cannot tweak my notion-app color with notion-enhancer

Debugging

  • Potential problem: Electron?

  • try installing notion using this https://github.com/jaredallard/notion-app instead of the AUR package.

  • potential problem: notion-enhancer does not have permission to modify notion-app color scheme

fixed

  • so the second problem is actually the reason that we see no change in the color scheme
  • For my case, i reinstall notion-enhancer using npm install, then change folder permission
sudo chmod -R a+wr /usr/bin/notion-app
sudo chmod -R a+wr /opt/notion-app/app
